## Running the Solutions

Most solutions are written in C++. To compile and run:

```bash
# Navigate to the day's directory
cd dayXX

# Compile (requires C++11 or later)
g++ -std=c++17 partY.cpp -o partY

# Run with input
./partY < input.txt
```
